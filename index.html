User Guide

Changing Output Value in K-Map

  • If the value of the cell is 0 or 1: Left-clicking on it flips the value (i.e., changes 0 to 1 or 1 to 0) and right-click changes the value to x.
  • If the value of the cell is x: Left-clicking on it changes the value to 0 and right-click changes the value to 1
  • Click and Drag: After clicking on a cell, hold the mouse button and you can drag that changed value to the neighbouring cells.

Changing Output Value in Truth Table

  • All the elements in the "Out" column of the truth table are buttons, clicking on a button cycles the value through 0, 1, and x (in that order).
  • K-Map and Truth Table are synchronized, so any changes made in one will automatically be reflected in the other.

Changing Variable Names

  • The elements of the header row are text edit boxes, click on a box and type a character to change the variable name.
  • Variable names are limited to a single character. If multiple characters are entered, only the last character will be used.
  • Variable names must be uppercase alphabetic characters. Lowercase entries will be automatically converted to uppercase.
  • If the newly entered variable name already exists, the names of the two variables will be swapped.

Changing the Number Of Variables

  • Use the dropdown menu located at the top right corner of K-Map.

Solving K-Map

  • Click the "Solve K-Map" button or press the "Enter" key on your keyboard.
